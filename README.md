# 💰 Loan Default Prediction App

ระบบทำนายการผิดนัดชำระหนี้ด้วย Machine Learning (SVM)

## 📊 ข้อมูลที่ใช้

ข้อมูลสินเชื่อจริงประกอบด้วย:
- ข้อมูลส่วนบุคคล (อายุ, เพศ, การศึกษา, รายได้)
- ข้อมูลสินเชื่อ (จำนวนเงินกู้, วัตถุประสงค์, อัตราดอกเบี้ย)
- ข้อมูลเครดิต (คะแนนเครดิต, ประวัติเครดิต)
